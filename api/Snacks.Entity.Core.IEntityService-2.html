An action to perform against a scoped instance of TDbContext
Remarks

Use this method if you need to perform more than one CRUD operation against a scoped instance of TDbContext within the same transaction.

Examples
entityService.AccessDbContext(dbContext => {
    dbContext.AddRange(entities);
    dbContext.SaveChanges();
});
